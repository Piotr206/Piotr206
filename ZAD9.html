    2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie, jako podłączenie stylu CSS?

    Wystarczy wpisać w nagłówku w każdym dokumencie na pozycji head, a znacznik wygląda tak:
    < link rel="Stylesheet" type="text/css" href="style.css"/ >
    taki link można dołączyć dowolną ilość razy - < link rel="Stylesheet"/ >.

    3. Jak jest najprostsza forma wstawiania stylu zewnętrznego -> otoczony ramką (może być tabela jedno komórkowa?

    < link rel="Stylesheet" type="text/css" href="style.html" />

    4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

    Można dołączyć dowolną liczbę zewnętrznych arkuszy stylów - każdy jako osobny element <  link rel="Stylesheet" />

    5. Co się dzieje w przypadku konfliktów? Przez konflikt rozumiemy, gdy te same style są definiowane w różnych arkuszach styli podłączone do tego samego arkusza.

    przy konfliktach ważniejsze są deklaracje z arkusza dołączonego później

    6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    tworzone są w zwykłym pliku tekstowym, do ich tworzenia i edytowania wystarcza zwykły edytor tekstu

    7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    pliki tworzone i edytowanie z użyciem specjalnego edytora CSS są wygodniejsze oraz szybsze do tworzenia.

    8. Jakie rozszerzenie posiada plik będący zewnętrznym arkuszem styli?

    takie pliki mają rozszerzenie ".css"

    9. Jakie są zalety zewnętrznego arkusza styli( dwie – z użyciem wyliczenia?

    Zalety zewnętrznego arkusza styli:
    1.wszystkie strony połączone mogą mieć wspólne cechy

    10. Uwagi na temat pisania w CSS.

    Uwagi na temat zewnętrznych CSS:
    -Należy pamiętać, że plik będący zewnętrznym arkuszem stylów musi mieć rozszerzenie *.css
    -Znacznik LINK może znajdować się tylko i wyłącznie w nagłówku dokumentu.(sekcja head )
    -W przypadku konfliktów, ważniejsze będą deklaracje z arkusza dołączonego później. -Zewnętrzny arkusz stylów jest on zwykłym plikiem tekstowym. Do tworzenia i edytowania wystarczy zwykły edytor tekst.
    -Można się posłużyć specjalnym edytorem CSS.
    -Można dołączyć dowolną liczbę zewnętrznych arkuszy stylów - każdy jako osobny element < link rel="Stylesheet" />.
